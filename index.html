Статодинамика по Селуянову

ИЗОТОН для 50+ | Гормональный акцент
Параметры статодинамики
  • Вес: 20-40% от 1RM (5-12 кг)
  • Темп: 3 сек вверх + 3 сек вниз
  • Подход: 30-40 сек → до жжения
  • Серия: 3 подхода, отдых 30 сек
  • Между сериями: 5-7 мин
  • Амплитуда: частичная, мышца не расслабляется
  • Дыхание: НИКОГДА не задерживать!
Безопасность
  • BB < 30 → тонизирующая (1-2 серии)
  • BB < 20 → отдых, без тренировки
  • Пульс цель: 90-110 bpm
  • Плечи: амплитуда до 45°, без жима стоя
  • Колени: частичная амплитуда, без прыжков
Ноги первыми = гормоны
  • Каждая тренировка начинается с приседаний
  • Ноги выделяют 4x больше гормонов чем руки
  • 1 серия приседаний → тестостерон + GH
  • Затем целевые мышцы "захватывают" гормоны
